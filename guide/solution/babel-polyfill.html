# 使用 @babel/polyfill 兼容IE

vueJs只兼容ie9及以上的浏览器

element-ui兼容ie10及以上的浏览器

部分es6语法是不被ie所兼容的，为了解决这个，我们可以使用@babel/polyfill

# 安装

npm install babel-polyfill -s
1

# 使用

首先在main.js中引用如下代码：

import '@babel/polyfill'
1

然后修改babel.config.js为：

module.exports = {
  presets: [
    ['@vue/app', {
      useBuiltIns: 'entry'
    }]
  ]
}
