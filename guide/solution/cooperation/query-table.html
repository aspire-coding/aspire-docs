# 关于query-table 可搜索表格组件的前后端同学协作

# 请求字段

  • 当前页： page_no
  • 页数： page_size
  • 排序字段： prop
  • 排序方式： order

例：

{
  page_no: 1,
  page_size: 20,
  prop: 'created_at',
  order: 'ascending'  // ascending or descending
}

  • 表格数组：result
