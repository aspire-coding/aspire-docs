# Query-Table

# 介绍

目前query table 由三个小组件组成

  • tool-bar
  • query-form
  • table-list

这三个组件是query-table的核心，它分别是功能按钮组件、搜索组件、表格组件。组合在一起时可以快速的生成一个表格。但是并不是一定要组合在一起才能使用， 根据业务场景拆开来使用也是一种不错的选中。

优势

当团队逐渐变大时，就不得不考虑几个问题。代码规范、ui规范。使用 query-table 我们不用去考虑这个问题。甚至你完全不用去考虑 ui的层面，你只需要考虑数据，只要数据对了，生成的页面就是正确的。

# HTML

<template>
    <div>
      <query-table :tools="tools"
                   :tables="tables"
                   :form-fields="formFields"></query-table>
